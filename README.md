# CloudWatch Monitoring Project

This repository contains:
- CloudWatch Agent configuration file
- Commands used to install and run CloudWatch Agent on EC2
- Screenshots related to CloudWatch setup (optional)
Real-Time EC2 System Log Monitoring Using CloudWatch
Implemented an end-to-end monitoring system on AWS by installing and configuring the CloudWatch Agent on an EC2 instance to stream SSH authentication logs into CloudWatch Logs. Created a custom log file, validated agent configuration, and enabled real-time log monitoring. Demonstrated skills in Linux, IAM roles, EC2 administration, and AWS CloudWatch.
